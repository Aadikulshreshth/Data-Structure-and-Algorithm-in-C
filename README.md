# Data Structures and Algorithms in C

This repository contains implementations of fundamental Data Structures and Algorithms written in C as part of academic coursework and independent practice. The programs focus on understanding algorithm design, traversal techniques, sorting methods, and core data structure operations.

## Topics Covered

### Searching

* Linear Search
* Binary Search
* Breadth First Search
* Depth First Search

### Sorting

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort

### Linked Lists

* Singly Linked List
* Doubly Linked List
* Circular Linked List

### Trees and Graphs

* Tree Traversal
* Graph Representation
* Dijkstra’s Algorithm

## Language

C

## Compilation and Execution

Example:

gcc filename.c -o output
./output

Replace `filename.c` with the required source file.

## Purpose

The goal of this repository is to build a strong foundation in Data Structures and Algorithms through hands-on implementation and experimentation in C.
